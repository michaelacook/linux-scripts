# Linux Scripts
This repo contains the scripts I use to install software and manage my system. Currently the main scripts I have are for installing all the base packages and applications that I always use on a daily driver machine, as well as scripts to install yay AUR helper.

I run Arch, BTW. (Had to, sorry)

## Caveats
Ensure that the script `install-aur-pacakges.sh` is run from your home folder and not inside the repo folder. For example:

```bash
cd ..
./linux-scripts/install-aur-packages.sh
```
